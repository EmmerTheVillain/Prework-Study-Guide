Git

  • Enter your Git notes here
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A adds the new working branch created to be staged.
  • git commit -m "'comment'" commits the changes.
  • git push origin branch-name pushes the changes to the repo
  • git pull will pull the most updated version from the repo
